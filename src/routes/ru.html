Добавление дат в bookmarks


    Для добавления дат в файл закладок на браузере, экспортируйте файл закладок и сохраните на диске (например для браузера chrome выберите настройки - ︙ в правом верхнем углу ⇒ Bookmarks ⇒ Bookmark manager или CTRL + SHIFT + O. Далее ︙ в правом верхнем углу на сайте ⇒ Bookmarks ⇒ Export bookmarks). Файл будет иметь название типа "bookmarks_XX.XX.XXXX.html" (X - цифра). Далее нажмите кнопку "Загрузите файл" и выберите сохраненный файл закладок. В папке Downloads на вашем компьютере появится файл "bookmarks-result.html" с датами. Это может занять до одной минуты
